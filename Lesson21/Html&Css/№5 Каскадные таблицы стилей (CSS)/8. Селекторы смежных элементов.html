Селекторы элементов одного уровня

Селекторы элементов одного уровня или смежных элементов позволяют выбрать элементы, которые находятся на одном уровне вложенности.

Иногда такие элементы еще называют сиблинги (siblings).

Если необходимо стилизовать вообще все смежные элементы одного уровня, то в этом случае необходимо использовать знак тильды "~".
